# Задача 1: Первый HTTP‑сервер

## Цель

Закрепить базовые вещи из лекции:

- запуск простого HTTP‑сервера на Go;
- работу с `http.HandleFunc`;
- проверку ответа через браузер или `curl`.

## Условие

Реализуй самый простой HTTP‑сервер:

- сервер слушает порт `8080`;
- по адресу `/` (корень) возвращает текст **`Hello Bitlab!`**.

После запуска сервера ты должен увидеть этот текст:

- в браузере по адресу `http://localhost:8080/`;
- или в терминале через `curl`.

## Требования к реализации

1. Использовать пакет `net/http`.
2. Зарегистрировать хендлер для пути `/` с помощью `http.HandleFunc`.
3. В хендлере вернуть строку `Hello Bitlab!` с помощью `fmt.Fprint` или `fmt.Fprintf`.
4. Запустить сервер с помощью `http.ListenAndServe(":8080", nil)`.

## Шаблон кода

Создай файл `main.go` в папке `lesson1/practice/task_1` и используй следующий шаблон:

```go
package main

import (
    "fmt"
    "net/http"
)

func main() {
    // TODO: зарегистрировать хендлер для "/"

    // TODO: запустить сервер на порту 8080
}
```

## Как проверить решение

1. Запусти сервер из папки `lesson1/practice/task_1`:

   ```bash
   go run main.go
   ```

2. Открой в браузере:

   - `http://localhost:8080/`

3. Или используй `curl`:

   ```bash
   curl "http://localhost:8080/"
   ```

4. Убедись, что в ответе приходит текст:

   ```text
   Hello Bitlab!
   ```
