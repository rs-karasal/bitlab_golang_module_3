# Bitlab Golang — Модуль 3

Учебный репозиторий для 3-го модуля курса **«Golang»** школы **Bitlab**.  
Здесь собраны:

- наглядные примеры кода, разбираемые на занятиях;
- практические задачи для самостоятельной отработки темы;
- небольшие мини-проекты, помогающие связать теорию с реальной разработкой;
- заготовки и шаблоны, которые можно использовать как основу для своих приложений.

Репозиторий задуман как «песочница» для студентов: место, где можно пробовать, ломать, исправлять и учиться на живом Go-коде.

## Как работать с материалами

1. Клонировать репозиторий.
2. Открыть интересующую тему/задачу.
3. Запустить и модифицировать код, экспериментируя с решениями.
4. Сравнить своё решение с предложенными вариантами (если есть).
5. Задавать вопросы и предлагать улучшения.
